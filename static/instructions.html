Dashboard Guide

How to get the most out of the AI Model Dashboard.

üîç Global Search

The global search bar in the header is your gateway to the entire ecosystem. It allows you to find models across all categories (LLMs, Image, Video, Voice) instantly.

  • Smart Prefetching: Start typing, and the dashboard automatically connects to external catalogs (Fal.ai, Replicate, Testing Catalog) to find fresh results.
  • Deep Linking: Clicking a result instantly navigates to the correct tab and scrolls the item into view.

‚öñÔ∏è Compare System

Compare models side-by-side to make informed decisions. This feature works best when comparing models within the same category (e.g., LLM vs LLM), but supports cross-category comparison for pricing and speed attributes.

How to use:

  1. Click the Compare button on any model card.
  2. The Compare Tray will appear at the bottom of the screen.
  3. Add up to 5 models.
  4. Click Open Comparison on the tray to view the detailed table.

üíæ Saved Views

Don't lose your filter context. You can save specific combinations of filters for quick access later.

Workflow:

  • Set your filters (e.g., "OpenRouter models provided by Microsoft").
  • Click Save view in the section header.
  • Name your view (e.g., "Microsoft Models").
  • Restore it anytime from the Views... dropdown.

‚ö° Power Features

Monitor Tab

A curated feed of updates from specific high-signal sources (Discord, X). Good for catching breaking news before it hits the main feeds.

Hype Tab

Tracks "trending" repos and models based on community engagement (stars, forks, mentions). Use the toggle in the Latest tab to blend Hype data into your daily feed.

Agent Mode

The Agent tab accepts natural language ("Show me cheap coding models") and attempts to control the dashboard to show you the answer.
